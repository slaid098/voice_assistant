# Голосовой помощник

Голосовой ассистент для слабовидящих людей. Управление YouTube, погода, таймер, текущее время — голосом на русском языке. Звуковые ярлыки для accessibility.

## Возможности

- **YouTube поиск** — голосовой запрос → озвучка результатов → навигация (дальше/назад/включить)
- **Название видео** — «что играет?» → название текущего видео
- **Погода** — текущий прогноз для любого города
- **Время** — текущие дата и время
- **Таймер** — голосовой таймер с озвучкой истечения
- **Звуковые ярлыки** — бип перед записью, чайм при поиске, фанфара при старте, финал после команды

## Установка

### Из исходников

```bash
uv sync --extra dev
uv run voice-assistant
```

### Конфигурация

Скопируйте `.env.template` в `.env` и заполните:

| Переменная | Описание | По умолчанию |
|-----------|----------|-------------|
| `WAKE_WORD` | Слово активации | `вики` |
| `WAKE_THRESHOLD` | Порог fuzzy-матчинга (0-100) | `70` |
| `WAKE_TIMEOUT_MS` | Таймаут ожидания активации (мс) | `3000` |
| `COMMAND_TIMEOUT_MS` | Таймаут ожидания команды (мс) | `6000` |
| `WEATHER_DEFAULT_CITY` | Город по умолчанию | `Гомель` |
| `OPENWEATHER_API_KEY` | Ключ OpenWeather API | (обязательно) |
| `YOUTUBE_SEARCH_LIMIT` | Количество результатов поиска | `10` |

## Команды

Скажите слово активации, затем команду:

- **Ютуб [запрос]** — поиск видео на YouTube
- **Погода [город]** — прогноз погоды
- **Время** — текущие дата и время
- **Таймер [N] минут/секунд/часов** — поставить таймер
- **Название** — что сейчас играет
- **Помощь** — список команд
- **Стоп** — завершить
